================================================================================
O QUE A AUTOMACAO FAZ
================================================================================

Exporta relatorios de Riscos Social, Ambiental e Climatico (RSAC) do portal
Sisbr 3.0 para cada cooperativa configurada, preenche uma planilha consolidada
mensal e publica tudo no SharePoint. Ao final, envia e-mail de resumo.

================================================================================
ARQUITETURA
================================================================================

Padrao: Dispatcher/Performer (orquestrado pelo Jarbis/Camunda)

Dispatcher (task_dispatcher.py):
  1. Le Config.xlsx (aba Items — 1 linha por cooperativa)
  2. Cria 1 item no banco (PostgREST) por cooperativa
  3. Verifica duplicatas antes de inserir

Performer (task_performer.py):
  Para cada cooperativa:
  1. Abre Sisbr 2.0 Desktop (lib_sisbr_desktop — login automatico)
  2. Acessa modulo "RISCOS SOCIAL, AMBIENTAL E CLIMATICO"
     -> Sisbr abre Chrome com sessao autenticada (certificado digital)
  3. Navega no portal web RSA (https://portal.sisbr.coop.br/rsa/risco):
     - Clica em "Relatorios" -> "Relatorios de Riscos Social Ambiental e Climatico"
     - Preenche: Tipo Relatorio (Relatorio por Cooperativa), Singular ({cooperativa}), Mes/Ano
     - Clica "Exportar" -> seleciona formato XLSX -> "Gerar Relatorio"
     - Aguarda "Relatorios Disponiveis", valida situacao, clica "Imprimir"
  4. Salva arquivo via dialogo "Salvar como" do Windows (pywinauto)
  5. Le o relatorio exportado (insumo)
  6. Copia TODO o conteudo do insumo para a aba da cooperativa no consolidado mensal
  7. Upload do insumo no SharePoint (pasta da cooperativa)
  8. Upload do consolidado atualizado no SharePoint (pasta do mes)
  9. Ao final de TODOS os itens, envia e-mail de resumo com template Sicoob

================================================================================
POR QUE NAO USA SELENIUM
================================================================================

O portal RSA exige sessao autenticada no Sisbr com certificado digital.
Selenium cria sandbox limpa (sem cookies/sessao). A automacao usa pyautogui +
pywinauto para interagir com o Chrome ja aberto pelo Sisbr, injetando
JavaScript via barra de endereco para localizar elementos no DOM e obter
coordenadas de tela para clicar.

================================================================================
CONSOLIDADO MENSAL
================================================================================

- 1 arquivo unico por mes: RSAC_{MMYYYY}.xlsx
- Copiado do template Models/Modelo_PlanilhaPrincipal.xlsx na 1a cooperativa
- Template tem 136 abas (4 por cooperativa + 8 gerais)
- Abas da cooperativa no template vem VAZIAS (podem ter merged cells)
- A automacao copia TODO o conteudo do insumo (titulo, data emissao, criterios,
  headers e dados) para a aba correspondente
- Antes de copiar, desfaz merges e limpa conteudo existente
- Upload incremental: a cada cooperativa, o consolidado atualizado sobe pro SharePoint

================================================================================
UTILS REUTILIZAVEIS
================================================================================

Para replicar em outro projeto RPA Sicoob:
1. Copiar pasta utils/ (ou subpacotes necessarios)
2. Copiar lib_sisbr_desktop/ se precisar do Sisbr Desktop
3. Configurar .env com credenciais e PROJECT_NAME
4. Usar:
   from utils.mail import send_mail, build_status_email
   from utils.sharepoint import upload_file
   from utils.database import ensure_project, insert_item
5. O template de e-mail se adapta automaticamente via project_name e intro
